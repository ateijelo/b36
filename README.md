## Convert from and to base36 (at the same time)

### Installation

Just run:

    sudo make install

### Usage

Running `b36 ff` gives:

    base 10   base 36
    -------   -------
         ff         -
        555        ff

And running `b36 555` gives:

    base 10   base 36
    -------   -------
        555        ff
       6665       555

That's it.
