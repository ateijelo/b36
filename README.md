## Convert from and to base36 (at the same time)

### Installation

Just run:

    sudo make install

### Usage

    $ b36 ff
    base 10   base 36
    -------   -------
         ff         -
        555        ff

    $ b36 555
    base 10   base 36
    -------   -------
        555        ff
       6665       555

That's it.
